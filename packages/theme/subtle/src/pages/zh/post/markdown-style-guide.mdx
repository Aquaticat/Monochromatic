  ## Blockquotes The blockquote element represents content that is quoted from another source, optionally with a
  citation which must be within a `footer` or `cite` element, and optionally with in-line changes such as annotations
  and abbreviations.

  <s>
    ### Blockquote without attribution

    <s>
      #### Syntax ```markdown > Tiam, ad mint andaepu dandae nostion secatur sequo quae. > **Note** that you can use
      _Markdown syntax_ within a blockquote. ```
    </s>
    <s>
      #### Output > Tiam, ad mint andaepu dandae nostion secatur sequo quae. > **Note** that you can use _Markdown
      syntax_ within a blockquote.
    </s>
  </s>

  <s>
    ### Blockquote with attribution

    <s>
      #### Syntax ```markdown > Don't communicate by sharing memory, share memory by communicating.<br>
      > — <cite>Rob Pike[^1]</cite>
      ```
    </s>
    <s>
      #### Output > Don't communicate by sharing memory, share memory by communicating. > > — <cite>Rob Pike[^1]</cite>

      [^1]: The above quote is excerpted from Rob Pike's [talk](https://www.youtube.com/watch?v=PAAkCSZUG1c) during
      Gopherfest, November 18, 2015.
    </s>
  </s>
</s>

  ## List Types

  <s>
    ### Ordered List

    <s>
      #### Syntax ```markdown 1. First item 2. Second item 3. Third item ```
    </s>

    <s>
      #### Output 1. First item 2. Second item 3. Third item
    </s>
  </s>

  <s>
    ### Unordered List

    <s>
      #### Syntax ```markdown - List item - Another item - And another item ```
    </s>

    <s>
      #### Output - List item - Another item - And another item
    </s>
  </s>

  <s>
    ### Nested list

    <s>
      #### Syntax ```markdown - <p>Fruit</p>
      - Apple - Orange - Banana - <p>Dairy</p>
      1. Milk 2. Cheese ```
    </s>
    <s>
      #### Output - <p>Fruit</p>
      - Apple - Orange - Banana - <p>Dairy</p>
      1. Milk 2. Cheese
    </s>
  </s>
</s>
