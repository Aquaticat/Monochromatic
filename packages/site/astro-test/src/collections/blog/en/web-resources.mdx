## Web Typography by Richard Rutter (Book £14.95 + VAT) - Must read

[Web Typography - A handbook for designing beautiful and effective responsive typography](https://book.webtypography.net/)

Companion website: [The Elements of Typographic Style Applied to the Web](https://webtypography.net/)

## A Better Way to Learn JavaScript (Book series $99.95) - Recommended reference material, quick cheatsheets

[A Better Way to Learn JavaScript Series | Leanpub](https://leanpub.com/b/a-better-way-to-learn-javascript-book-series)

## JavaScript.Info (Website + Forum free oss) - Must read

[The modern JavaScript Tutorial](https://javascript.info)

## You Don’t Know JS (Book series free oss) - Alternate tutorial to JavaScript.Info, unfinished

[You Don’t Know JS](https://github.com/getify/You-Dont-Know-JS)

## Functional-Light JavaScript (Book free oss) - Not essential, but I highly recommend reading it at least once, just to get an idea what functional programming is.

[Functional-Light JavaScript](https://github.com/getify/Functional-Light-JS)

## Inclusive Components (Book €18) - Must read

[Inclusive Components | Heydon Pickering](https://book.inclusive-components.design/)

## Every Layout (Book $69) - Must read

[Every Layout - Relearn CSS layout by example](https://every-layout.dev/)

## Material Design (Design System free oss) - Must read

[Material Design | Design System](https://m3.material.io)

## Figma on Youtube (video playlists free) - Recommend checking out interesting ones

[Figma | Youtube](https://www.youtube.com/@Figma/playlists)

## Heydon Pickering (personal website of a great frontend dev) - Recommend checking out and subscribing to.

[HeydonWorks](https://heydonworks.com/)

## CUBE CSS (CSS methodology free oss) - Must read

[Composition Utility Block Exception Cascading Style Sheets](https://cube.fyi/)

## adactio - Jeremy Keith (personal website of a great frontend dev) - Recommend checking out

[adactio - Jeremy Keith](https://adactio.com)

## James’ Coffee Blog (personal website of a great frontend dev) - Recommend checking out

[James’](https://jamesg.blog)

## Nick Punt (personal website of a great frontend designer) - Recommend checking out

[Nick Punt](https://nickpunt.com/)

## Ahmad Shadeed (personal website of a great frontend designer/dev) - Must read

[Ahmad Shadeed](https://ishadeed.com/)

## Professor Frisby’s Mostly Adequate Guide to Functional Programming (Book online free oss) - Only read if you want to brush up on functional programming

[gitbook](https://mostly-adequate.gitbook.io/mostly-adequate-guide)
